# xhico.github.io

Personal portfolio site for **Francisco Filipe** — Senior Backend Engineer specializing in Python, distributed systems, and high-scale data platforms.

🌐 **Live:** [xhico.github.io](https://xhico.github.io)

## About

Single-page site showcasing experience, skills, and services. Built for hiring managers and B2B clients evaluating me for fully remote backend engineering roles or contract work.

## Stack

- Plain HTML, CSS, and a tiny bit of vanilla JS
- [Inter](https://fonts.google.com/specimen/Inter) and [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) via Google Fonts
- Inline SVG favicon — no build step, no dependencies
- Deployed via [GitHub Pages](https://pages.github.com/) from the `main` branch

## Files

| File                               | Purpose                                               |
|------------------------------------|-------------------------------------------------------|
| `index.html`                       | The site — all markup, styles, and scripts inline     |
| `favicon.svg`                      | Cyan `F` monogram on dark navy                        |
| `Francisco_Filipe_CV.pdf`          | Downloadable CV linked from hero and contact sections |
| `Francisco_Filipe_CoverLetter.pdf` | Cover letter PDF                                      |

## Local preview

Open the file:

```bash
open index.html
```

Or serve it with any static server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
